# Analiza sentymentu tweetów z wykorzystaniem SHAP

Projekt realizowany w ramach przedmiotu **Metodyki Inteligencji Obliczeniowej**.

Celem projektu jest zbudowanie modelu analizy sentymentu dla danych tekstowych (tweetów) z wykorzystaniem metod przetwarzania języka naturalnego (NLP), a następnie interpretacja działania modelu przy użyciu metody SHAP (SHapley Additive exPlanations).

W projekcie wykorzystywany jest zbiór danych:
https://www.kaggle.com/datasets/austinreese/trump-tweets
